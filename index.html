Professional Self-Assessment

Throughout my time in the Computer Science program at Southern New Hampshire University, I have developed technical skills, problem-solving abilities, and a stronger understanding of how computer science concepts are applied in real-world situations. My coursework and project work helped me grow in software engineering, algorithms and data structures, databases, security, and technical communication. This ePortfolio represents that growth by bringing together artifacts that demonstrate both my academic progress and my ability to improve and refine existing systems.

One of the most valuable parts of this program was learning how to approach software development as both a technical and collaborative process. Through my coursework, I strengthened my ability to communicate with different audiences, including instructors, users, and stakeholders. I learned that strong communication is essential when explaining technical decisions, presenting project goals, documenting systems, and refining solutions based on feedback. In several projects, I had to think not only about whether a program worked, but also whether it was understandable, maintainable, and appropriate for the intended audience.

My work in software engineering helped me develop a stronger appreciation for structure, organization, and maintainability in code. Early in my studies, I focused mainly on making programs function correctly. Over time, I learned that effective software design also requires modularity, readability, validation, and thoughtful planning. This growth is reflected in my enhancement to the mobile weight-tracking application, where I improved the code structure, reorganized database operations, and strengthened input validation to make the application more reliable and maintainable.

I also developed stronger skills in algorithms and data structures by working with projects that required efficient data retrieval, filtering, and processing. I learned how data organization and algorithmic decisions directly affect performance and usability. In my enhancement of the Animal Shelter Dashboard, I improved the project by adding better validation, sorting, and query optimization. This experience reinforced the importance of designing systems that are not only functional, but also efficient and scalable when working with larger datasets.

Database development was another important area of growth throughout the program. I gained experience working with persistent data, organizing information, and building applications that rely on structured storage and retrieval. I learned the importance of data integrity, reliable database interactions, and user-focused functionality. Across my projects, I applied these principles in ways that improved both usability and long-term maintainability.

Security also became a more important part of how I approach software development. As I progressed through the program, I began to think more carefully about validation, safe handling of user input, protection of data, and reducing vulnerabilities through better design. Even when working on projects that were not solely focused on cybersecurity, I learned that a strong security mindset should always be part of the development process. Adding validation, improving login-related logic, and considering data integrity in my projects helped reinforce this perspective.

The artifacts included in this ePortfolio work together to demonstrate the range of skills I developed in the program. The mobile weight-tracking application highlights software design and engineering principles. The Animal Shelter Dashboard demonstrates my understanding of algorithms, data structures, and efficient data handling. The OpenGL Graphics Project reflects my ability to improve an interactive system through better rendering behavior, lighting controls, and user interaction. Together, these artifacts show not only the final results of my work, but also my ability to evaluate existing systems, identify areas for improvement, and implement meaningful enhancements.

Overall, this program has prepared me to enter the computer science field with a stronger technical foundation, greater confidence in my problem-solving abilities, and a better understanding of professional expectations in software development. This ePortfolio reflects both the skills I have built and the progress I have made, and it represents the kind of thoughtful, organized, and adaptable work I hope to continue producing in my future career.

Artifacts

Enhancement One: Mobile Weight Tracker

This artifact is a mobile weight-tracking application originally created in CS 360. I selected this project for my portfolio because it demonstrates my ability to build a practical mobile application that combines user interaction, persistent data storage, and core software engineering principles. The original project allowed users to create accounts, log in, and manage weight entries through create, read, update, and delete functionality.

For this enhancement, I focused on software design and engineering improvements. I refactored the project to improve modularity by moving database-related responsibilities into a dedicated helper class rather than handling too much functionality directly within the activity logic. I also improved input validation, strengthened data integrity, and made the application more user-friendly by improving error handling and overall interaction flow. These changes made the project more maintainable, more organized, and more reliable for users.

This enhancement reflects my growth in writing cleaner code, organizing responsibilities more effectively, and designing with both functionality and long-term maintainability in mind. Through this process, I strengthened my understanding of software design principles, debugging, validation, and user-centered development.

Enhancement Two: Animal Shelter Dashboard

This artifact is a client-server dashboard application originally developed in CS 340. I chose this project because it demonstrates my ability to work with data-driven systems, database queries, and dynamic user interaction. The original application allowed users to work with animal shelter data through a custom CRUD module and an interactive dashboard that supported filtering and rescue-related analysis.

For this enhancement, I focused on algorithms and data structures. I improved the project by adding stronger validation to query handling, implementing sorting and result limiting, and improving the overall efficiency of data retrieval and processing. These changes made the project more scalable and more effective when working with large datasets, while also improving readability and maintainability within the code.

This enhancement demonstrates my ability to apply algorithmic thinking to practical software problems. It also reflects my growth in designing systems that are not only functional, but efficient, structured, and adaptable to future changes. Through this work, I gained a stronger understanding of how algorithm and data structure decisions affect performance and usability in real applications.

Enhancement Three: OpenGL Graphics Project

This artifact is a computational graphics project originally created in CS 330. I selected this project because it highlights my ability to work with interactive systems, rendering logic, and visually oriented software design. The original project presented a 3D graphics scene built with OpenGL and demonstrated core concepts in camera movement, lighting, and scene construction.

For this enhancement, I improved the project by adding features that increased both functionality and interactivity. These changes included improved rendering behavior, lighting mode controls, camera reset capabilities, and better interaction between application components. By refining how the system handled lighting and user input, I made the project more flexible, visually effective, and easier to navigate.

This enhancement reflects my ability to analyze and improve a more complex interactive system while maintaining organization across multiple source files. It also demonstrates my growth in graphics programming, modular design, and program refinement. Through this work, I strengthened my understanding of user control, rendering quality, and the importance of structuring code clearly in larger systems.
